## üó∫Ô∏è The Path

### [1. HTML5 (HyperText Markup Language)](./01_HTML5_Deep_Dive.md)
*The Skeleton*
- **Tags to know**: `div`, `span`, `h1`-`h6`, `p`, `a`, `img`, `ul/li`, `form`, `input`.
- **Semantic HTML**: Using `<nav>`, `<main>`, `<footer>` instead of just `<div>` everywhere.

### [2. CSS3 (Cascading Style Sheets)](./02_CSS_Mastery.md)
*The Skin & Clothes*
- **Box Model**: Margin, Border, Padding, Content. (This is crucial!).
- **Flexbox**: Aligning items in rows/columns easily. covering `justify-content`, `align-items`.
- **Grid**: 2-dimensional layouts.
- **Responsive Design**: Using `@media` queries to make sites look good on mobile.

### [3. JavaScript (The DOM)](./03_JS_DOM_Interaction.md)
*The Muscles/Movement*
- **Selection**: `document.getElementById()`, `querySelector()`.
- **Events**: `onClick`, `onSubmit`, `onScroll`.
- **Manipulation**: Changing text, colors, or classes when an event happens.
